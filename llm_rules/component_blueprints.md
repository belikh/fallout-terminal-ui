# Component Blueprints

## 1. Vault Boy Animation Engine (`fallout-vault-boy`)
A specialized component for rendering animated Vault Boy SVGs based on system state.

### Animations
- `idle`: Breathing animation.
- `walking`: Leg and arm cycle.
- `thumbs_up`: Static pose with glowing effect.
- `alert`: Shaking/Jitter effect for critical states.

## 2. RobCo Terminal Suite (50+ Card Types)
A comprehensive library of Lovelace cards mapping to Vault-Tec systems.

### Diagnostic Modules
- `fallout-mainframe-card`: Central processor status.
- `fallout-power-card`: Grid monitoring.
- `fallout-rad-card`: Radiation detection.
- ... (and 47 others)

### Control Modules
- `fallout-switch-card`: Toggle logic.
- `fallout-lock-card`: Security door controls.
- `fallout-override-card`: Manual system takeover.
